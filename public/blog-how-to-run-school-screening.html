How to run a school screening program

Practical guidance to run ethical, high-coverage school vision screenings.

Step-by-step

  • Plan dates with schools and education authorities.
  • Train teachers/volunteers on visual acuity testing.
  • Use simple referral forms and parent consent letters.
  • Arrange follow-up clinics for spectacles and suspected pathology.
